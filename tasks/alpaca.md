# Alpaca

```bash
flatpak install flathub com.jeffser.Alpaca
```

As of April 2026, there is no need to download Ollama separately. Just then go through the in-app onboarding process for Ollama. 
During the onboarding process, under the screen titled **Instances**, click the **Create Ollama Instance** button, and then click **Install Ollama**. It's about 1.9 GB in size.

Ollama installation ends up at `~/.var/app/com.jeffser.Alpaca/data/ollama_installation`

Ollama models end up in `~/.var/app/com.jeffser.Alpaca/data/.ollama/models/blobs`

Use Flatseal to disable network, after you acquired desired models.

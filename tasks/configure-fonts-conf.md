# fonts conf

```bash
mkdir ~/.config/fontconfig 
nano ~/.config/fontconfig/fonts.conf
ln -s ~/.config/fontconfig/fonts.conf ~/.fonts.conf
```

then logout and log back in

Tested on:

- Ubuntu 24.04 LTS
- Ubuntu 26.04 LTS

See also: https://manpages.ubuntu.com/manpages/resolute/man5/fonts-conf.5.html
